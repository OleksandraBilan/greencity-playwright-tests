# Playwright Test Framework

## Опис проєкту

Цей проєкт є базовою структурою тестового фреймворку для автоматизованого тестування вебзастосунку з використанням Playwright та TypeScript.

Проєкт підготовлений для подальшої реалізації автотестів з використанням Page Object Model та component-based підходу.
